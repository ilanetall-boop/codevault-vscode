# Changelog

All notable changes to CodeVault will be documented in this file.

## [0.1.0] - 2024

### Added
- Initial release
- Three memory types: semantic (facts), episodic (events), procedural (how-tos)
- Project analyzer for automatic stack/architecture detection
- Auto-capture for git commits, file changes, and terminal commands
- Memory sidebar with search functionality
- Context export for AI agents (Ctrl+Shift+E)
- Manual memory capture (Ctrl+Shift+R)
- Memory recall with search (Ctrl+Shift+M)
- Status bar indicator
- Offline queue for when backend is unavailable
- Support for Windows, macOS, and Linux
